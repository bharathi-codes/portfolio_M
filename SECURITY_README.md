# Security Notice

This directory contains sensitive documents (resume and certificates) that are protected and not included in version control.

**For deployment:**
- Upload these files directly to your hosting service
- Do not commit to public repositories
- Ensure proper file permissions on server

**Protected Files:**
- Resume PDF
- Certificate images

**Protection Methods:**
1. Files excluded from Git via .gitignore
2. Client-side protection (right-click disabled)
3. Hotlink protection via referrer check
4. Download prevention on website
5. Dev tools disabled on resume page
